Código-fonte / Pré-formatação

Quando for escrever linguagens de programação ela fica sem espaçamento: document.getElementByid:('teste')


Para melhorar usamos a tag < code > ficando assim : document.getElementByid:('teste)

Os codigos escritos não ficam em ordem : num = int(input('digite um numero')) if num % 2 ==0: print(f'O numero {num} é par') else: Print(f'O número {num} é impar) print('Fim do programa')

Para que as palavras fiquem na ordem escrita , usamos a tag < pre > : 

    num = int(input('digite um numero'))
        if num % 2 ==0:
           print(f'O numero {num} é par')
      else:
         Print(f'O número {num} é impar)
      print('Fim do programa')
